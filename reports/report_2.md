## Tribonacci sequence
https://www.codewars.com/kata/tribonacci-sequence
```
module Tribonacci where

tribonacci :: Num a => (a, a, a) -> Int -> [a]
tribonacci (a, b, c) n = reverse $ tri n [c, b, a]

tri 0 _ = []
tri 1 l = drop 2 l
tri 2 l = drop 1 l
tri 3 l = l
tri n l = tri (n-1) (sum ( take 3 l) : l)
```
